# Introduction

[Dora](https://github.com/dora-rs/dora) is a multilingual framework for running real-time multi-AI and multi-hardware applications.

Dora by Examples is a series of runnable examples designed to help users quickly get started and learn how to use the Dora framework. These examples cover various application scenarios, including the use of various sensors, image classification, object detection, speech recognition, motor control, and more. Examples are provided in Python, Rust, C++, and other languages.
