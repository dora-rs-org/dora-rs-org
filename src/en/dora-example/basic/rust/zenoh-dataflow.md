# Zenoh Dataflow

This example demonstrates how to use Rust with Zenoh for distributed dataflow communication.

[Zenoh](https://zenoh.io/) is a high-performance distributed communication protocol that supports publish/subscribe, query/response patterns, suitable for edge computing and IoT scenarios.
