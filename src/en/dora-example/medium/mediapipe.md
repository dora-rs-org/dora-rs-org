# Pose Recognition

> Location: ./examples/mediapipe/
> Keywords: mediapipe, pose recognition

This example captures images through `opencv-video-capture`,
extracts pose points using `dora-mediapip`,
and finally visualizes pose points and images through `dora-rerun`

# UV Environment, Install Related Nodes
```bash
$ dora build rgb-dev.yml --uv
```

# Run
```bash
$ dora run rgb-dev.yml --uv
```
