# Dora-nav 导航

基于开源机器人中间件DORA的导航框架，该导航框架基于激光点云实现环境地图构建与室内/室外定位，采用轨迹跟踪的方法实现路径跟随导航。

激光雷达使用RoboScience Helios 16，小车底盘使用mickrobot 四轮差速小车。

**目录说明**：在"~"目录下存放dora 运行文件及 DORA_NAV文件夹

## 功能特点

- 激光点云环境地图构建
- 室内/室外定位
- 轨迹跟踪路径跟随导航
- 多模态数据可视化
