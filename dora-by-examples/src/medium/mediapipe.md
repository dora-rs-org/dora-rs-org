# 姿态识别

> 地址：./examples/mediapipe/
> 关键词：mediapipe，姿态识别

本样例通过`opencv-video-capture`获取图像，
`dora-mediapip`进行姿态点提取，
最后通过`dora-rerun`可视化姿态点与图像

# UV 环境，安装相关Node
```bash
$ dora build rgb-dev.yml --uv
```

# 运行
```bash
$ dora run rgb-dev.yml --uv
```
